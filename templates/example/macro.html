{% macro fruit_list(fruits) %}
    {% for fruit in fruits %}
  • {{ fruit }}
    • {% endfor %}
{% endmacro %}

과일 목록

{{ fruit_list(fruits) }}

  • Jinja의 매크로는 하나의 함수로, HTML 문자열을 반환한다.
  • 매크로 사용의 주요 목적은 하나의 함수를 사용해 반복적으로 작성하는 코드를 줄이는 것이다.

{#macro 정의#} {% macro input(name, value='', type='text', size=20) %}
{% endmacro %} {#macro 적용#} {{input('username')}} {{input('email')}} {{input('password', type='password')}}
